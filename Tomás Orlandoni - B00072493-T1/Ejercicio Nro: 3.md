# Ejercicio Nro: 3

# Enunciado
¿Qué ventajas y desventajas tienen los diagramas UML?

# Resolución

El Lenguaje Unificado de Modelado (UML) es el estándar de la industria para visualizar, especificar, construir y documentar los artefactos de un sistema de software.

## Ventajas
- Lenguaje universal: Lo entiende cualquier desarrollador del mundo, sin importar su idioma.
- Mejora la comunicación: Es mucho más fácil mostrarle un diagrama a un cliente o gerente que mostrarle código puro.
- Visión global: Permite ver la arquitectura completa y detectar fallos de diseño antes de empezar a programar.
- Excelente documentación: Es una guía visual invaluable para mantener el sistema a futuro o integrar nuevos miembros al equipo.

## Desventajas
- Lleva tiempo: Diseñarlos retrasa el inicio de la programación, lo que puede ser un problema en metodologías muy ágiles.
- Difíciles de mantener actualizados: Si el código cambia y nadie actualiza el UML, el diagrama queda obsoleto y se vuelve inútil.
- Curva de aprendizaje: Tiene demasiadas reglas y tipos de diagramas que hay que aprender a usar correctamente.
- Riesgo de detallar de más: Se puede perder muchísimo tiempo intentando dibujar cada pequeño detalle innecesario.
